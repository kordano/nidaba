# nidaba

Content Management System with Ring and Websocket Server.

## Usage

Not yet implemented.

## Deployment

Check latest leiningen version

```
lein upgrade
```

Build executable jar file

```
lein uberjar
```

Run jar file

```
java -jar target/nidaba-0.1.0-standalone.jar
```
